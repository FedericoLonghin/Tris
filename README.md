# Tris
A terminal based tris version!

## How to play
Run ` git clone https://github.com/FedericoLonghin/Tris && cd Tris && ./Tris ` for instantly playing the game.

## Gamemode:
- Multiplayer (doesn't need an explanation)
- Singleplayer (you play againsta th AI)
